Kyrksök startade på hack4heritage i oktober 2016 och byggdes av laget Holy Hit bestående av Albin Larsson, Jan Ainali, Lars Lundqvist, Ulrika Nilsson och David Zardini.

Kyrksök bygger på data från Wikidata, Bebyggelseregistret, Wikipedia, och Wikimedia Commons. I framtiden planeras även data från Svenska kyrkans API att hämtas in.

Bilderna från Wikimedia Commons kan ha lite olika licensvillkor. För full attribuering och licensinformation, klicka på bilden. Texten från Wikipedia är under CC BY-SA 3.0. Klicka på Wikipedialänken och sedan historikfliken för att se alla bidragsgivare.

Hjälp gärna till att utveckla projektet på Github eller diskutera i Facebookgruppen om du känner dig mer bekväm där.
